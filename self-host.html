ğŸš€ Quick Start Deploymentâ€‹

1. Docker Compose Deploymentâ€‹

Prerequisites:

# Install Docker and Docker Compose
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER
sudo curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose

Deployment:

# Clone repository
git clone https://github.com/aicser-platform/aicser-world
cd aicser-world

# Configure environment
cp env.example .env
nano .env  # Edit with your configuration

# Start services
docker-compose up -d

# Check status
docker-compose ps

Configuration:

# Key environment variables
AI_PROVIDER=openai
OPENAI_API_KEY=your_api_key_here
JWT_SECRET=your_jwt_secret_here
POSTGRES_PASSWORD=secure_password_here

2. Production Configurationâ€‹

Security Hardening:

# Generate strong secrets
openssl rand -hex 32  # JWT secret
openssl rand -hex 32  # Encryption key

# Set file permissions
chmod 600 .env
chmod 600 ssl/*.pem

SSL Configuration:

# Obtain SSL certificate (Let's Encrypt)
sudo certbot certonly --standalone -d your-domain.com

# Configure NGINX with SSL
cp nginx/nginx.conf.example nginx/nginx.conf
# Edit nginx.conf with your domain and SSL paths

Database Setup:

# Create production database
docker-compose exec postgres psql -U aicser_user -d postgres
CREATE DATABASE aicser_platform_prod;
GRANT ALL PRIVILEGES ON DATABASE aicser_platform_prod TO aicser_user;
